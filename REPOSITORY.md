## About

This repository is a fork/copy of the Vercel platform monorepo, containing:
- Vercel CLI tools
- Build utilities and packages
- Runtime packages
- Examples and templates
- Testing infrastructure

## Quick Start

```bash
# Install dependencies
pnpm install

# Build the project
pnpm build

# Run tests
pnpm test
```

## Development

This project uses:
- **pnpm** for package management
- **Turbo** for monorepo builds
- **TypeScript** for type safety
- **Jest** for testing

For more information, see the main [README.md](./README.md).
