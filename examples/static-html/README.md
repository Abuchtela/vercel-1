# Static HTML Example

This directory is a simple example of a static HTML website that can be deployed to Vercel with zero configuration.

## Features

- Pure HTML, CSS, and JavaScript
- No build tools or dependencies required
- Interactive button demonstrating client-side JavaScript
- Modern, responsive design
- Ready to deploy in seconds

## Deploy Your Own

Deploy your own Static HTML project with Vercel.

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/vercel/vercel/tree/main/examples/static-html&template=static-html)

## Getting Started Locally

Since this is a static HTML site, you can simply open the `index.html` file in your browser:

```bash
# Option 1: Open directly in browser
open index.html

# Option 2: Use a simple HTTP server
python -m http.server 8000
# or
npx serve
```

Then open [http://localhost:8000](http://localhost:8000) in your browser.

## Deploying From Your Terminal

You can deploy your Static HTML project with a single command using [Vercel CLI](https://vercel.com/download):

```shell
vercel
```
